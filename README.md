

# Install


1. Clone repo into computer
2. Run in cmd or git 
```bash
npm i 
```
3. Config .env

   - In .env has
   
      -DB_HOST= host db
      
      -DB_USER= username db
      
      -DB_PASS= pass db
      
      -COLLECTION_NAME= collection name
      
      -KEY_SESSION_SECERET= key seceret
      
4. Run project by
```bash
npm start 
```
